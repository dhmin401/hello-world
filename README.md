# Weather
## Overview
Weather web app displays current weather of selected city or province. It uses HTML, CSS and JavaScript.

## Feature
1. It displays details of current weather.
- Temperature
- Humidity
- Pressure
- Wind
- Clouds

2. It uses autocomplete for searching location.
